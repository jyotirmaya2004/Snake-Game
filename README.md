# 🐍 Snake Game

A modern **Snake Game** built using **HTML, CSS, and JavaScript** with smooth animations, random food images, score tracking, timer, and local storage support for high score.

---

## 🎮 Features

- Classic snake gameplay
- Smooth snake movement
- Animated food items
- Random food images
  - 🍎 Apple
  - 🍕 Pizza
  - 🍔 Burger
  - 🍇 Grapes
  - 🍩 Donut
- Live score counter
- High score stored using **Local Storage**
- Game timer (`MM:SS`)
- Start and restart modal
- Smooth transitions and animations
- Responsive game board

---

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## 🎯 Controls

Use your keyboard arrow keys:

- ⬆️ `Arrow Up` → Move Up
- ⬇️ `Arrow Down` → Move Down
- ⬅️ `Arrow Left` → Move Left
- ➡️ `Arrow Right` → Move Right

---

## 📊 Game Info

The game includes:

- **Current Score**
- **Highest Score**
- **Live Timer**
- **Restart Option**

High score is saved in browser local storage, so it remains even after refreshing the page.
